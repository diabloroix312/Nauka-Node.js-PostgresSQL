#  Witcher RPG Node.js Edition

To jest gra RPG inspirowana światem Wiedźmina, stworzona w ramach nauki Node.js i baz danych PostgreSQL.

## Funkcje projektu
* **System Walki:** Logika turowa z obliczaniem szans na unik i obrażenia.
* **Baza Danych:** Integracja z PostgreSQL (zapisywanie i wczytywanie stanu gry).
* **Zmienne Środowiskowe:** Bezpieczne połączenie z bazą przy użyciu `dotenv`.
* **Ewolucja Kodu:** Projekt rozwijany przez 20 dni nauki JavaScript.

## ⚙️ Jak uruchomić projekt
1. Upewnij się, że masz zainstalowany **Node.js** oraz **PostgreSQL**.
2. Sklonuj to repozytorium.
3. Stwórz plik .env i uzupełnij go swoimi danymi do bazy (użytkownik, hasło, host, port).
4. Uruchom grę node ProjektGra.js
